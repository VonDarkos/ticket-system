# Ticket System

A modern ticket management application built with React, TypeScript and Vite.

## Live Demo

https://my-ticket-system.netlify.app/

## Features

* Create new tickets
* Edit ticket status (Open, In Progress, Closed)
* Delete tickets
* Search tickets
* Filter by status and priority
* Ticket statistics dashboard
* Ticket detail page
* Image upload support
* Attachment links
* Persistent storage with localStorage
* Responsive design
* Custom 404 page

## Tech Stack

* React
* TypeScript
* Vite
* React Router
* Context API
* localStorage
* CSS

## Installation

```bash
git clone https://github.com/VonDarkos/ticket-system.git
cd ticket-system
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The application is deployed on Netlify.

React Router support is enabled through:

```txt
public/_redirects
```

```txt
/* /index.html 200
```
